INTRODUCTION
------------
Scald: Mixcloud is a audio provider allowing Scald module users
to add Mixcloud Atoms of type audio.

Scald is an innnovative take on how to handle Media Atoms in Drupal.

REQUIREMENTS
------------
This module requires the following modules:
* Scald (https://drupal.org/project/scald)

INSTALLATION
------------
* Install as usual, see
https://drupal.org/documentation/install/modules-themes/modules-7
for further information.
